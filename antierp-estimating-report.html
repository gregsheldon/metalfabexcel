      What to fix first

      These first moves create the most value because they reduce the places where quoting errors and margin leaks usually begin.

      1. Standardize labour build-ups so hours are built by task, not guessed from memory.
      2. Add a missed-scope checklist before every quote leaves the shop.
      3. Separate shop labour from install labour so field work is not buried in one number.
      4. Add forced checks for coating, trucking, rentals, hardware, anchors, and subcontractors.
      5. Build a final quote review dashboard that shows margin before submission.
      6. Auto-populate proposal and handoff outputs so the estimate carries through to the customer and shop floor.

      Manager discussion questions

      Use these questions with your estimator, shop manager, project manager, or business partner. The answers will usually reveal where the estimating system needs to be tightened first.

      Process questions

      • Where do we usually miss scope?
      • Which quote types take the longest?
      • Are labour hours being built up or guessed?
      • Do we have one controlled workbook or several versions?

      Margin questions

      • Do we know margin before sending every quote?
      • Are install assumptions separate from shop labour?
      • Are rentals, trucking, coating, and hardware checked every time?
      • Can someone else quote the same job consistently?
